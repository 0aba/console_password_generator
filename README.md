# Описание
Приложение предоставляет возможности по генерации паролей, а так же разные другие действия с паролями например, проверка его надежности.
Приложение разрабатывалось для учебной практики по Git.

## Содержание
1. [Требования для работы](#Требования-для-работы)
2. [Зависимости Python](#Зависимости-Python)

## Требования для работы
- **Python** ~=3.13.0
